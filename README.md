# NanoCem UT-9 — анимированные SVG-ассеты (EASYMIX)

Готовые самостоятельные SVG со встроенной анимацией (CSS внутри `<svg>`) и поддержкой
`prefers-reduced-motion`. Анимация работает при вставке через `<img>`, `<object>` или
инлайном. Палитра и шрифты — фирменные EASYMIX.

## Подключение

**Анимация сохраняется** при таком встраивании:

```html
<!-- через img (CSS-анимация внутри SVG работает) -->
<img src="assets/img/hero-particles.svg" alt="Проникновение частиц NanoCem UT-9">

<!-- favicon -->
<link rel="icon" type="image/svg+xml" href="assets/img/favicon.svg">
```

Для интерактивных версий (hover, слайдер масштаба, счётчики в графиках по скроллу)
вставляйте SVG **инлайном** в HTML и управляйте через JS из брифа.

## Цвета (для согласования с CSS сайта)

`--ink #1A1C1A` · `--paper #F5F6F3` · `--accent #6E7F6A` · `--muted #8A8F88`
· `--paper-2 #ECEEEA` · `--line #D6D9D3`

## Шрифты

В SVG заданы `Space Grotesk` (заголовки), `JetBrains Mono` (метки/цифры) с системными
fallback. При инлайн-вставке подхватятся ваши Google Fonts; при вставке через `<img>`
используется системный запасной шрифт — это нормально.

## Примечания

- `hero-particles.svg` рассчитан на тёмный фон hero (`--ink`). Остальные — на светлый
  (`--paper`), фон прозрачный.
- Графики кейсов отрисовывают линии и метки по загрузке; для «докрутки» чисел-счётчиков
  по скроллу используйте инлайн-версию + JS.
- Все цифры выверены по ТУ и фактическим данным Каратюбе. Терминология — каноническая
  (без «тампонажный/squeeze/кварц/песок», только нефтяные скважины, ссылки на ТУ).
